# random-acg-image-api
随机二次元图片

### API 端点：

#### 横屏图片：

- **请求方式**：GET
- **URL**：https://api.rulian.us.kg/horizontal
- **图片数量**：500张
- **分辨率**：最小 4K，最大 8K
- **描述**：返回适用于横屏显示的高清ACG图片，分辨率范围从4K到8K，适合大屏幕展示。

#### 竖屏图片：

- **请求方式**：GET
- **URL**：https://api.rulian.us.kg/vertical
- **图片数量**：500张
- **分辨率**：最小 2K，最大 8K
- **描述**：返回适用于竖屏显示的高清ACG图片，分辨率范围从2K到8K，适用于手机、平板等设备。

#### 自适应设备图片：

- **请求方式**：GET
- **URL**：https://api.rulian.us.kg/adaptive
- **描述**：自动检测设备的屏幕方向，提供相应的横屏或竖屏图片。

### 图片来源：
全部图片来自Pixiv（P站），都是我自己一张一张找的并且部分超分。
